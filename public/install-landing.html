üîí Secure Alternative (Verify First)

Download, review, then execute:

Download Installation Package

wget https://kcclanka.com/gigies/gigies-bot-latest.tar.gz tar -xzf gigies-bot-latest.tar.gz cd gigies-bot-*/

Run Installation Script

chmod +x install.sh sudo ./install.sh

Follow Interactive Prompts

You'll be asked for:

  • Customer slug (your store name)
  • Port number (default: 3011)
  • Domain name (optional)
  • Nginx configuration (yes/no)
  • Ollama API key (optional, for AI)

Access Your Dashboard

http://YOUR-SERVER-IP:PORT/dashboard.html Default Login: Username: admin Password: admin123
